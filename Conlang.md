### 1.2 Vowels
#### 1.2.1 monophthong
|           | **Front**    | **Back** |
| --------- | ------------ | -------- |
| **Close** | i uy [y]     | eu [ɯ] u |
| **Mid**   | e [ɛ] oy [œ] | eo [ʌ] o |
| **Open**  | a [ɐ]        |          |

This is the listed version of the vowels:\
a, i, eu, e, eo, uy, u, oy, o
